# Python Projects

A collection of Python projects showcasing various applications and utilities built with Python.

## 📁 Projects Included

This repository contains the following projects:

### 1. **Image Resizer**
A Python utility for resizing images with ease. This tool helps you batch resize images or resize individual images to your desired dimensions.

**Features:**
- Easy image resizing
- Batch processing support
- Maintains aspect ratio options

---

### 2. **PDF Merger**
A convenient tool to merge multiple PDF files into a single document. Perfect for combining related documents, reports, or presentations.

**Features:**
- Merge multiple PDF files
- Maintain original formatting
- Simple and intuitive interface

---

### 3. **RoboSpeaker**
A text-to-speech application that converts written text into spoken audio. Useful for accessibility, automation, and content consumption.

**Features:**
- Text-to-speech conversion
- Multiple voice options
- Audio output in various formats

---

### 4. **Weather App**
A weather application that provides current weather information and forecasts. Get real-time weather updates for any location.

**Features:**
- Current weather information
- Weather forecasts
- Search by location
- Temperature, humidity, and wind speed data

---
